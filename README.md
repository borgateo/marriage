# My Marriage - Static Site with Ruby on Heroku/Cedar using a [Rack](http://rack.rubyforge.org/) app.

Project organised like this:

```
- marriage
  |- config.ru
  |- Gemfile
  |- public
    |- index.html
    |- images
    |- js
    |- css
```

1. `$ bundle install`
1. `$ npm install`
1. build: `$ gulp`  watch: `$ gulp watch`
1. `$ rackup` localhost:9292
